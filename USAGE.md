# 股票監控系統 - 使用說明

## 🚀 快速開始

### 方法一：使用啟動腳本 (推薦)

雙擊執行 `start-dev.bat` 檔案，會自動開啟兩個終端機視窗：
- 視窗 1: 後端伺服器
- 視窗 2: 前端伺服器

### 方法二：使用 npm 指令

在專案根目錄執行：
```bash
npm run dev
```

這會同時啟動前後端伺服器。

### 方法三：手動分別啟動

**終端機 1 - 啟動後端：**
```bash
cd server
npm run dev
```

**終端機 2 - 啟動前端：**
```bash
cd client
npm run dev
```

## 📱 系統使用

### 1. 存取系統

啟動完成後，在瀏覽器開啟：
- **前端介面**: http://localhost:5173
- **後端 API**: http://localhost:3001

### 2. 新增自選股

1. 點擊「自選股」分頁
2. 點擊「+ 新增」按鈕
3. 輸入股票資訊：
   - 股票代號 (例: 2330)
   - 股票名稱 (例: 台積電)
   - 選擇市場 (台股/美股)
4. 點擊「確認新增」

**範例股票代號：**
- 台股: 2330 (台積電), 2317 (鴻海), 2454 (聯發科)
- 美股: AAPL (Apple), GOOGL (Google), MSFT (Microsoft)

### 3. 查看股票分析

1. 在自選股清單中點擊任一股票
2. 右側會顯示：
   - 即時報價 (延遲 15-20 分鐘)
   - AI 快速摘要
   - 技術分析
   - 籌碼分析

### 4. 設定警報

1. 點擊「警報設定」分頁
2. 點擊「+ 新增警報」按鈕
3. 選擇警報條件：
   - 股價突破: 當股價超過目標價時觸發
   - 股價跌破: 當股價低於目標價時觸發
   - 成交量爆量: 成交量異常放大時觸發
   - 突破近期高點: 創新高時觸發
   - 外資連買: 外資連續買超時觸發
   - 黃金交叉: 短期均線突破長期均線
   - 死亡交叉: 短期均線跌破長期均線

### 5. 管理投資組合

1. 點擊「資產配置」分頁
2. 點擊「+ 新增持股」
3. 輸入持股資訊：
   - 股票代號和名稱
   - 持股張數
   - 買入成本價
   - 目標配置比例 (%)
4. 查看總覽：
   - 總市值
   - 持股檔數
   - 總損益
5. 點擊「📊 計算再平衡」查看調整建議

## 🔧 常見問題

### Q: 為什麼股價不是即時的？
A: 台股即時報價需要付費授權，系統使用的免費 API 有 15-20 分鐘延遲，但足夠用於學習和分析。

### Q: 可以交易嗎？
A: 本系統僅提供分析功能，不支援下單交易。

### Q: 美股報價準確嗎？
A: 美股使用 Yahoo Finance API，報價相對準確，但非官方來源。

### Q: 如何停止伺服器？
A: 在終端機視窗按 `Ctrl+C` 即可停止。

### Q: 資料存在哪裡？
A: 所有資料存在 `server/data/stock-monitor.db` SQLite 資料庫中。
