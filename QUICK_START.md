# 🚀 快速開始指南

## 30 秒快速啟動

### Windows 用戶

1. **雙擊執行啟動腳本**
   ```
   start-dev.bat
   ```

2. **等待啟動完成** (約 5-10 秒)

3. **開啟瀏覽器**
   ```
   http://localhost:5173
   ```

就這麼簡單！✨

---

## 第一次使用

### 步驟 1: 新增第一檔自選股

1. 點擊「自選股」分頁
2. 點擊「+ 新增」按鈕
3. 輸入：
   - 股票代號: `2330`
   - 股票名稱: `台積電`
   - 市場: `台股`
4. 點擊「確認新增」

### 步驟 2: 查看股票分析

1. 在自選股清單點擊「台積電 (2330)」
2. 右側會顯示：
   - 📊 即時報價
   - 🎯 AI 快速摘要
   - 💡 投資建議

### 步驟 3: 設定價格警報

1. 點擊「警報設定」分頁
2. 點擊「+ 新增警報」
3. 選擇條件：
   - 股票代號: `2330`
   - 條件: `股價突破`
   - 目標值: `1800`
4. 系統會在達到條件時提醒您

### 步驟 4: 管理投資組合

1. 點擊「資產配置」分頁
2. 點擊「+ 新增持股」
3. 輸入您的持股資訊
4. 點擊「📊 計算再平衡」查看建議

### 技巧 1: 建立觀察清單

## ❓ 遇到問題？

### 無法啟動？
1. 確認 Node.js 已安裝 (版本 18+)
2. 確認已執行 `npm install`
3. 檢查 3001 和 5173 端口是否被占用

### 看不到股價？
- 台股報價延遲 15-20 分鐘是正常的
- 非交易時段無法取得最新報價
- 等待下個交易日開盤

### API 錯誤？
1. 確認後端伺服器已啟動
2. 檢查 http://localhost:3001/health
3. 查看終端機錯誤訊息
