## 🔌 API 端點總覽

### 自選股 API
```
GET    /api/watchlist           # 取得所有自選股
POST   /api/watchlist           # 新增自選股
DELETE /api/watchlist/:symbol   # 刪除自選股
```

### 報價 API
```
GET    /api/quotes/:symbol              # 取得即時報價
GET    /api/quotes/:symbol/history      # 取得歷史資料
```

### 警報 API
```
GET    /api/alerts       # 取得所有警報
POST   /api/alerts       # 新增警報
PATCH  /api/alerts/:id   # 更新警報狀態
DELETE /api/alerts/:id   # 刪除警報
```

### 持股組合 API
```
GET    /api/portfolio            # 取得所有持股
GET    /api/portfolio/details    # 取得詳細資訊
GET    /api/portfolio/rebalance  # 取得再平衡建議
POST   /api/portfolio            # 新增持股
PUT    /api/portfolio/:id        # 更新持股
DELETE /api/portfolio/:id        # 刪除持股
```

**API 測試**: 全部通過 ✅

## ⚠️ 重要注意事項

### 資料來源限制
1. **台股報價延遲**: 證交所免費 API 有 15-20 分鐘延遲
2. **非官方 API**: Yahoo Finance 為非官方 API，可能隨時變更
3. **無即時資料**: 不支援真正的即時報價 (需付費授權)

### 功能限制
1. **僅供學習**: 本系統為教育用途，不構成投資建議
2. **不可交易**: 無下單功能，僅供分析使用
3. **資料準確性**: 建議與官方資料交叉驗證

### 系統需求
- Node.js 18 或以上版本
- Windows / macOS / Linux
- 現代瀏覽器 (Chrome, Edge, Firefox)
- 網路連線
